# Patrones de piel en dermatologia
Patrones de piel en dermatología aplicando como area: Aprendizaje no Supervisado y subarea:Visión Por Computadora 
## Vision del sistema
Aplicación diseñada para apoyar el diagnóstico temprano de enfermedades dermatológicas mediante inteligencia artificial. El sistema combina aprendizaje no supervisado, clustering difuso y visión por computadora para analizar imágenes de la piel, identificar patrones ocultos y clasificar lesiones con alta precisión.

Este proyecto forma parte de la asignatura de Sistemas Colaborativos y busca integrar técnicas modernas de IA en aplicaciones médicas.

## Flujo del sistema
1. Entrada: Imágenes dermatológicas.
2. Preprocesamiento: Normalización, corrección de color, eliminación de artefactos.
3. Segmentación: Identificación de la región de interés.
4. Extracción de características: Color, textura, forma, bordes y características profundas (CNN).
5. Reducción de dimensionalidad: PCA, t-SNE o UMAP.
6. Clustering: K-Means, DBSCAN, GMM o jerárquico.
7. Evaluación: Silhouette Score, Davies-Bouldin Index, Calinski-Harabasz Index.
8. Interpretación: Identificación de patrones comunes y posibles anomalías.
